# Meine Internetseite

Persönliche Portfolio-Website von mir — Data Analyst aus Düsseldorf.

## Inhalt

- **Übersicht** — Kurzprofil und Kontakt
- **Berufliche Laufbahn** — Stationen bei AXA, Henkel u.a.
- **Zertifikate** — DataCamp, Google, ExcelHero
- **Bildungsweg** — Studium an der HHU Düsseldorf
- **Skills** — Tools und Technologien (Power BI, SQL, Python, …)
- **About** — Hintergrund zur Website selbst

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- Bootstrap 5
- Chart.js (Diagramme)
- Hover.css (Animationen)

## Entwicklung

### Voraussetzungen

- [Node.js](https://nodejs.org) (für Prettier & HTMLHint)
- [Python](https://www.python.org) + [pre-commit](https://pre-commit.com)

### Setup

```bash
npm install
pre-commit install
```

### Code-Qualität

Vor jedem Commit laufen automatisch:

- **Prettier** — Formatierung (HTML, CSS, JS)
- **HTMLHint** — HTML-Strukturprüfung
- **pre-commit-hooks** — Trailing Whitespace, LF-Zeilenenden
